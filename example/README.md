# Better Sheet Example App

This is a demo application showcasing the Better Sheet component.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on your platform of choice:
```bash
# iOS
npm run ios

# Android
npm run android

# Web
npm run web
```

## What's Included

The example app demonstrates:
- Basic bottom sheet with multiple snap points
- Scrollable content with long lists
- Small quick-action sheets
- Different backdrop configurations
- Custom styling

## Testing the Component Locally

This example uses the local version of better-sheet via `"better-sheet": "file:.."` in package.json. Any changes you make to the parent directory will be reflected here after restarting the development server.
